Atalhos {
Copiar linha:
    Shift + Alt + ↓/↑

Mover linha:
    Alt + ↓/↑

Indentar linha:
    Ctrl + ]/[

Copiar tudo:
    Ctrl + A

Recortar:
    Ctrl + X

Selecionar várias linhas de uma vez:
    Alt + Click

Crase:
    Alt + 9 + 6
}

Operadores:
    + === adição
    - === subtração
    * === multiplicação
    / === divisão real (vírgula)
    % === divisão inteira (resto)
    ** === potência (ao quadrado)

    =   Atribui a
    ==	Igual a (pode ser string ou número)
    === Igual a (não pode ser string)
    !=	Diferente 
    <=	Menor ou Igual
    >=	Maior ou Igual


Switch só funciona quando não tem que comparar valores numéricos!
