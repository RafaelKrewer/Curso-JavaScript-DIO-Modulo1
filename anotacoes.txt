Copiar linha:
    Shift + Alt + ↓/↑

Mover linha:
    Alt + ↓/↑

Indentar linha:
    Ctrl + ]/[ 

Copiar tudo:
    Ctrl + A

Recortar:
    Ctrl + X

Selecionar várias linhas de uma vez:
    Alt + Click

=   Atribui a
==	Igual a (pode ser string ou número)
=== Igual a (não pode ser string)
!=	Diferente 
<=	Menor ou Igual
>=	Maior ou Igual
